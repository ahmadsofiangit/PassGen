Passgen is a wordlist generator. Allowing you to customize the level of password from medium, strong, and weak. This tool doesn't require installation of libraries; you have it ready to play with it.
Donation Support: https://buymeacoffee.com/retrobreado

### Installation:
``` shell
git clone https://github.com/ahmadsofiangit/PassGen.git
```

### Usage:
``` shell
  cd PassGen
  python3 PassGen.py -l <Set the line length of password> -c <Set character length of password> -m {level}
```
